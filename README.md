# AgentSpend — AI Cost Monitor

Track your AI API spending across models, projects, and time.

## Features
- 📊 Real-time cost dashboard with interactive charts
- 🤖 Multi-model tracking (Claude, GPT, Gemini, DeepSeek)
- 📁 Per-project cost breakdown
- ⚠️ Budget alerts and spending warnings
- 💡 Built-in cost-saving tips
- 📤 JSON export for external analysis
- 🔒 All data stored locally (localStorage, no backend)

## Quick Start
1. Open `index.html` in any browser
2. Start adding AI sessions via "+ Add Session"
3. Watch your costs in real-time

## Tech
- Single HTML file, zero dependencies except Chart.js CDN
- Dark theme, responsive design
- localStorage persistence
- No tracking, no analytics, no server
